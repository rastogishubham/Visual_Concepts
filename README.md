# Solution to the Important Message Challenge by Visual Concepts

makefile: There is a makefile present, make all will be used to make the output file.
To make sure that ansi c was used, gcc is used with the -ansi and -pedantic flags.

imp_mess_soln.c: This file contains the solution to the problem. It uses Dijkstra's
alogrithm to find the shortest path. There is a do-while loop so that the user can 
input the data till it inputted correctly.

imp_mess_soln.h: This is the header file for the solution and contains all the function
definitions and preprocessor directives.
